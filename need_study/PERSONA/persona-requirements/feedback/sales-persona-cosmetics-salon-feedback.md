## 1. 전체 평가

현재 기획은 살롱 대상 화장품 영업에게 **거래처 관계와 방문 이력을 관리하는 개인 도구**로 가능성이 있습니다. 특히 미용실 원장, 실장, 디자이너별 성향과 제품 반응을 기록하는 용도라면 실무성이 높습니다.

다만 살롱 영업은 단순 거래처 관리가 아니라 **매장 관계도**, **교육/시연 이력**, **제품 소진 주기**, **클레임**, **신제품 도입 가능성**을 관리해야 합니다. 현재 기획은 이 부분이 부족합니다.

### 1.1 잘 맞는 부분

| 기획 요소 | 적합도 | 이유 |
|----------|--------|------|
| 거래처 정보 관리 | 높음 | 살롱별 원장/실장/디자이너 정보가 중요 |
| Off-the-record 메모 | 매우 높음 | 원장 성향, 브랜드 충성도, 가격 민감도는 공식 시스템에 남기기 어려움 |
| 필터 | 높음 | 방문 예정, 신제품 제안 대상, 교육 필요 매장을 찾아야 함 |
| 모바일 동기화 | 매우 높음 | 차량 이동 중 다음 매장 정보를 확인함 |
| Excel Import/Export | 보통 | 초기 거래처 이전과 보고용으로 필요 |
| 명함 OCR | 보통 | 원장/교육 담당자 등록에 유용 |

## 2. 기능별 실용성 평가

### 2.1 영업 데이터 관리(CRUD)

**실용성**: 높음

살롱 영업에서는 `company`와 `customer` 구분이 중요합니다. 미용실 한 곳 안에도 의사결정자가 여러 명입니다.

필요 필드:

| 구분 | 필드 |
|------|------|
| 살롱 | 매장명, 지역, 규모, 의자 수, 주력 시술, 거래 상태 |
| 인물 | 원장, 실장, 디자이너, 구매 담당, 영향력 |
| 제품 | 사용 중인 라인, 월 사용량, 경쟁 브랜드, 소진 주기 |
| 교육 | 시연일, 교육 참석자, 반응, 다음 교육 필요 |
| 이슈 | 클레임, 배송 문제, 가격 불만, 경쟁사 제안 |

**매장 방문 직후 5분 안에 열까?**  
네. 다음 매장으로 이동하기 전에 원장 반응과 다음 제안 제품을 바로 적어야 합니다.

### 2.2 Import/Export

**실용성**: 보통

기존 거래처 목록을 가져오는 Import는 필요합니다. Export는 월간 방문 현황이나 신제품 제안 대상 리스트로 쓸 수 있습니다.

다만 PDF Export보다는 `방문 이력 요약`, `교육 진행 현황`, `신제품 제안 리스트`가 더 유용합니다.

### 2.3 필터 시스템

**실용성**: 매우 높음

필요 필터:

- 이번 주 방문 예정 살롱
- 최근 60일 미방문 살롱
- 신제품 제안 가능성 높음
- 경쟁 브랜드 사용 중
- 교육/시연 요청 있음
- 클레임 후속 처리 필요
- 월 사용량 높은 매장
- 원장 관계 온도 낮음

현재 기획의 일반 필터는 유지하되, 살롱 영업용 저장 필터를 제공해야 합니다.

### 2.4 명함 OCR

**실용성**: 보통

원장 명함 등록에는 유용합니다. 하지만 살롱 영업에서는 명함보다 매장 방문 중 대화 내용, 시술 메뉴, 디자이너 구성 정보가 더 중요합니다.

### 2.5 Off-the-record 메모

**실용성**: 매우 높음

살롱 영업에서는 관계 정보가 매출입니다.

예시:

- 원장은 가격보다 교육 지원을 중요하게 봄
- 실장이 실제 제품 선택에 영향력 있음
- A 디자이너가 염색 라인에 관심 많음
- 경쟁사 리베이트 제안에 흔들림
- 클레임 이후 감정이 좋지 않음
- 월말보다 월초 방문을 선호

메모는 살롱 단위와 인물 단위로 따로 남겨야 합니다.

## 3. 기획서가 놓친 살롱 영업 페인포인트

### 3.1 매장 내 인물 관계도

미용실은 원장만 잡으면 끝나는 구조가 아닙니다. 실장이나 핵심 디자이너가 제품 사용을 좌우합니다.

추가 제안:

- 인물별 역할: Owner / Manager / Stylist / Assistant
- 영향력: High / Medium / Low
- 관계 온도: Warm / Neutral / Cold
- 선호 제품/시술

### 3.2 교육과 시연 이력

프로페셔널 제품은 써봐야 팔립니다. 교육 이력 관리가 필수입니다.

필드 제안:

```
training_records
  - salon_id
  - product_line
  - training_date
  - attendees
  - reaction
  - next_action
```

### 3.3 제품 소진 주기와 재주문 타이밍

살롱은 제품을 반복 구매합니다. 소진 시점을 놓치면 경쟁사 제품으로 넘어갑니다.

필요 기능:

- 제품별 월 사용량
- 예상 재주문 시점
- 마지막 주문일
- 경쟁 브랜드 전환 위험

### 3.4 차량 적재/샘플 관리

살롱 영업자는 차량에 샘플, 카탈로그, 테스터를 싣고 다닙니다. 어떤 매장에 어떤 샘플을 전달했는지 기록해야 합니다.

## 4. 우선순위 제안

### Phase 1

- 살롱 카드
- 인물 카드
- 방문 메모
- 제품 사용 라인
- 빠른 검색/필터

### Phase 2

- 교육/시연 이력
- 샘플 전달 기록
- 재주문 예상일
- 클레임 후속 처리

### Phase 3

- 명함 OCR
- 방문 리포트 Export
- 지도 기반 방문 동선

### Phase 4 이후 또는 제외

- 자동 주문
- 가격 자동 제안
- 본사 ERP 대체
- 자동 홍보 메시지

## 6. 핵심 요약(TL;DR)

살롱 화장품 영업에는 **매장 안의 사람 관계와 교육 이력을 기억하는 도구**가 필요합니다.

반드시 보강할 기능:

1. 살롱별 인물 관계도
2. 교육/시연 기록
3. 제품 소진 주기와 재주문 타이밍
4. 클레임 후속 처리
5. 샘플 전달 이력

현재 기획은 방향은 맞지만, 일반 거래처 관리에서 **살롱 운영 관계 관리**로 더 좁혀야 실사용성이 올라갑니다.
