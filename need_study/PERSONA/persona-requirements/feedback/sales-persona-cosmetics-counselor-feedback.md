# 방문 화장품 상담사 관점 피드백 - 영업 관리 프로그램 서비스 기획서

> **피드백 작성자**: 박지영 (11년차 방문 화장품 상담사, 단골 고객 300명 관리)
> **작성일**: 2026-05-22
> **대상 문서**: `01_서비스_기획서.md`

---

## 1. 전체 평가

현재 기획은 방문 화장품 상담사에게 **단골 고객 관리 앱**으로 꽤 잘 맞습니다. 상담사는 회사 앱이나 주문 시스템을 쓰지만, 실제로 매출을 만드는 정보는 고객의 피부 고민, 선호 제품, 샘플 반응, 생일, 가족 행사, 재구매 시점 같은 개인 메모에 있습니다.

다만 `거래처`, `딜`, `영업건`이라는 구조는 화장품 방문 판매 현장과 거리가 있습니다. 이 persona에게 필요한 단위는 **고객**, **피부 상태**, **사용 제품**, **샘플**, **재구매 주기**, **상담 이력**입니다.

### 1.1 잘 맞는 부분

| 기획 요소 | 적합도 | 이유 |
|----------|--------|------|
| 개인용 영업 관리 | 매우 높음 | 회사 주문 시스템과 별도로 나만의 고객 메모가 필요함 |
| 모바일 동기화 | 매우 높음 | 고객 집, 카페, 행사장, 이동 중에 바로 확인해야 함 |
| Off-the-record 메모 | 매우 높음 | 피부 고민, 예민한 성향, 가족 선물 취향은 개인 메모에 가까움 |
| 필터 | 높음 | 재구매 예정, 샘플 후 미구매, 생일 고객을 찾는 데 필요 |
| Excel Import | 보통 | 기존 단골 목록 이전에는 필요 |
| 명함 OCR | 낮음 | 고객이 명함을 주는 상황이 많지 않음 |
| PDF Export | 낮음 | 보고용보다는 상담 직전 확인이 더 중요 |

### 1.2 가장 중요한 전제

방문 화장품 상담사는 복잡한 CRM을 원하지 않습니다. 상담 중 고객 앞에서 오래 입력할 수 없고, 연령대가 높은 상담사도 많기 때문에 **큰 글씨, 빠른 입력, 고객명 검색, 재구매 알림**이 핵심입니다.

## 2. 기능별 실용성 평가

### 2.1 영업 데이터 관리(CRUD)

**실용성**: 매우 높음

고객 한 명을 저장할 때 단순 연락처보다 아래 정보가 더 중요합니다.

| 구분 | 필요한 정보 |
|------|-------------|
| 기본 | 이름, 연락처, 지역, 선호 연락 방식 |
| 피부 | 피부 타입, 고민, 알레르기, 피해야 할 성분 |
| 제품 | 현재 사용 제품, 구매일, 사용 주기, 선호 브랜드/라인 |
| 상담 | 마지막 방문일, 상담 내용, 반응, 다음 제안 제품 |
| 선물 | 생일, 가족 생일, 명절 선물 수요 |
| 민감 메모 | 가격 민감도, 설득 방식, 싫어하는 표현 |

**고객 방문 직전 5분 안에 열까?**  
네. 고객 집에 들어가기 전 "지난번에 뭐 샘플 드렸지?", "이분 피부가 건성이었나?"를 바로 봐야 합니다.

### 2.2 Import/Export

**실용성**: 보통

초기 고객 목록 이전에는 Excel Import가 중요합니다. 상담사들은 엑셀, 휴대폰 연락처, 수첩, 카카오톡에 고객이 흩어져 있습니다.

하지만 일상 사용에서는 Import/Export보다 `고객 검색`, `구매 주기 알림`, `샘플 반응 기록`이 더 중요합니다.

PDF Export는 우선순위가 낮습니다. 출력물보다 휴대폰 화면에서 크게 보이는 고객 카드가 더 필요합니다.

### 2.3 필터 시스템

**실용성**: 매우 높음

필터는 방문 화장품 상담사에게 매출을 만드는 기능입니다.

필요한 필터:

| 필터 | 사용 장면 |
|------|----------|
| 재구매 예정 | 클렌징/크림/세럼 소진 시점 연락 |
| 샘플 제공 후 미구매 | 샘플 반응 확인 |
| 생일/기념일 임박 | 선물세트 제안 |
| 최근 3개월 미구매 | 이탈 방지 연락 |
| 고가 라인 구매 경험 | 프리미엄 제품 제안 |
| 민감성 피부 | 신제품 제안 전 주의 |
| 선물 구매 고객 | 명절·어버이날 캠페인 대상 |

현재 기획의 "이번 달 구매 가능성 높은 사람"은 이 업종에서는 **재구매 시점이 가까운 고객**으로 구체화해야 합니다.

### 2.4 명함 OCR

**실용성**: 낮음

방문 상담에서는 명함보다 휴대폰 번호, 카카오톡, 주소, 가족 관계가 더 중요합니다. 명함 OCR은 후순위로 내려도 됩니다.

대신 필요한 기능:

- 연락처에서 고객 가져오기
- 카카오톡 이름과 실제 이름 별칭 저장
- 주소/방문 가능 시간 저장
- 고객 사진 또는 제품 사진 첨부

### 2.5 Off-the-record 메모

**실용성**: 매우 높음

이 기능이 제품의 핵심입니다. 회사 주문 시스템에 넣을 수 없는 정보가 매출을 만듭니다.

예시:

- "향 강한 제품 싫어함"
- "딸 결혼 준비 중, 선물세트 가능성"
- "남편이 가격에 민감함"
- "지난번 샘플은 따갑다고 함"
- "전화보다 문자 선호"
- "친구 모임에서 공동구매 가능"
- "겨울에 보습 제품 많이 구매"

메모에는 태그가 필요합니다. `건성`, `민감`, `선물`, `고가`, `샘플`, `재구매` 같은 태그가 검색되어야 합니다.

### 2.6 인증

**실용성**: 필수

고객 피부 정보와 주소, 가족 정보가 들어가므로 계정 보안은 중요합니다. 다만 상담사 입장에서는 로그인 절차가 너무 복잡하면 사용을 포기합니다.

권장:

- 간편 로그인
- 휴대폰 잠금과 연동
- 자동 로그아웃 옵션
- 데이터 백업

## 3. 기획서가 놓친 방문 화장품 페인포인트

### 3.1 재구매 주기 예측

화장품은 제품별 소진 주기가 다릅니다. 크림, 세럼, 클렌징, 선크림은 다시 연락해야 하는 시점이 다릅니다.

필요한 기능:

- 제품별 예상 소진일
- 마지막 구매일 기준 알림
- 재구매 성공/실패 기록
- 계절별 제품 추천 메모

### 3.2 샘플 관리

샘플은 방문 판매에서 매우 중요합니다. 샘플을 누구에게 줬는지 잊으면 후속 판매가 끊깁니다.

필드 제안:

```
sample_records
  - customer_id
  - product_name
  - given_date
  - reaction
  - converted_to_purchase
  - follow_up_date
```

### 3.3 큰 글씨와 단순 화면

방문 상담사는 연령대가 다양합니다. 작은 글씨, 복잡한 표, 많은 컬럼은 현장에서 쓰기 어렵습니다.

권장 UX:

- 고객명 검색창을 첫 화면에 크게 배치
- 오늘 연락할 고객 목록
- 재구매 임박 고객 목록
- 버튼은 3~5개 핵심만 노출
- 메모 입력은 음성 입력도 고려

### 3.4 주소와 방문 동선

방문 판매는 지역 동선이 중요합니다. 현재 기획의 `region`은 단순 필터에 머물러 있습니다.

추가 제안:

- 지역별 고객 묶음
- 오늘 방문할 고객 목록
- 지도 앱 연결
- 방문 가능 시간대 기록

## 4. 우선순위 제안

### Phase 1에 반드시 넣을 것

- 고객 카드
- 피부 타입/고민 필드
- 구매 제품과 구매일
- 재구매 예정 필터
- 상담 메모
- 모바일 검색

### Phase 2로 올릴 것

- 샘플 제공/반응 기록
- 생일/기념일 알림
- 연락 예정일
- 지역별 고객 필터

### Phase 3에 적합한 것

- Excel Import
- 제품 사진 첨부
- 음성 메모
- 지도 앱 연결

### Phase 4 이후 또는 제외

- 명함 OCR
- PDF Export
- 복잡한 딜 파이프라인
- 자동 판매 메시지 발송
- 회사 주문 시스템 대체

## 5. 사용 의향

| 조건 | 사용 의향 | 이유 |
|------|----------|------|
| 현재 기획 그대로 | 보통 | 고객 메모는 가능하지만 화장품 특화가 약함 |
| 피부/제품/재구매 관리 추가 | 적극 사용 | 실제 매출과 연결됨 |
| 화면이 복잡함 | 사용 어려움 | 현장에서 오래 입력할 수 없음 |
| 자동 메시지 중심 | 거부 가능 | 단골 관계가 상할 수 있음 |
| 월 1~3만원 | 가능 | 단골 재구매 몇 건이면 회수 가능 |

## 6. 핵심 요약(TL;DR)

방문 화장품 상담사에게 이 서비스는 **단골 고객의 피부·제품·재구매 주기를 기억해주는 개인 노트**가 되어야 합니다.

반드시 보강할 기능:

1. 피부 타입과 고민 기록
2. 제품별 구매일과 재구매 예정일
3. 샘플 제공 후 반응 추적
4. 생일/기념일 선물 수요 관리
5. 큰 글씨의 단순한 모바일 고객 카드

명함 OCR이나 PDF보다 중요한 것은 **방문 직전 10초 안에 고객 상태를 떠올릴 수 있는 화면**입니다.
