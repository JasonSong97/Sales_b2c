## 1. 전체 평가

산업재 영업 관점에서 현재 기획은 **가장 잘 맞는 타깃 중 하나**입니다. 산업재 영업자는 회사 ERP나 주문 시스템을 쓰지만, 실제 영업 성과를 좌우하는 정보는 현장 담당자 성향, 설비 상태, 소모품 교체 주기, 경쟁사 납품 상황, 급한 납기 요청 같은 개인 메모에 있습니다.

이 서비스는 산업재 영업자에게 `회사 시스템 옆에 붙는 개인 현장 CRM`으로 포지셔닝하면 설득력이 있습니다.

### 1.1 적합도 요약

| 기획 요소 | 적합도 | 이유 |
|----------|--------|------|
| 거래처 CRUD | 매우 높음 | 제조업 거래처와 담당자 관리가 핵심 |
| 모바일/PC 동기화 | 매우 높음 | 공장 방문 중 확인하고 사무실에서 정리 |
| Off-the-record 메모 | 매우 높음 | 현장 반응, 구매 담당 성향, 경쟁사 정보 기록 |
| 필터/검색 | 매우 높음 | 교체 주기, 견적 후속, 미방문 거래처 조회 |
| Excel Import/Export | 높음 | 기존 거래처 목록과 견적 관리에 필요 |
| 명함 OCR | 높음 | 공장 담당자/구매팀 명함 등록에 유용 |

## 2. 기능별 실용성 평가

### 2.1 영업 데이터 관리(CRUD)

**실용성**: 매우 높음

산업재 영업에서는 회사, 공장, 부서, 담당자가 복잡하게 얽혀 있습니다. 단순 `customer`보다 계층 구조가 필요합니다.

필요 데이터:

| 구분 | 필드 |
|------|------|
| 회사/공장 | 회사명, 공장명, 주소, 업종, 라인, 설비 |
| 담당자 | 구매, 생산, 설비, 품질, 안전 담당 |
| 품목 | 납품 품목, 단가, 사용량, 교체 주기 |
| 영업기회 | 견적 요청, 테스트, 샘플, 계약, 재구매 |
| 현장 메모 | 불만, 경쟁사, 납기 민감도, 결제 성향 |

**공장 방문 직후 5분 안에 열까?**  
네. 현장에서 "지난번에 어떤 베어링이 문제였지?", "이 라인은 어느 업체 제품 쓰고 있지?"를 바로 찾아야 합니다.

### 2.2 Import/Export

**실용성**: 높음

기존 거래처와 품목 데이터가 엑셀에 많습니다. Import는 도입 장벽을 낮춥니다. Export는 견적 대상, 방문 계획, 미수/후속 관리 목록으로 필요합니다.

PDF Export는 후순위입니다. 산업재 영업자는 PDF보다 Excel과 검색 가능한 모바일 화면을 더 많이 씁니다.

### 2.3 필터 시스템

**실용성**: 매우 높음

필요 필터:

- 최근 60일 미방문 거래처
- 견적 후 미회신
- 샘플 테스트 중
- 교체 주기 임박
- 경쟁사 사용 중
- 단가 인상 협의 필요
- 미수/결제 이슈
- 납기 클레임 발생

현재 기획의 "이번 달 구매 가능성 높은 사람"은 산업재에서는 **교체/재구매 시점이 가까운 거래처**로 바꿔야 합니다.

### 2.4 명함 OCR

**실용성**: 높음

공장 현장에서는 구매팀, 설비팀, 생산팀, 품질팀 담당자 명함을 계속 받습니다. OCR은 유용합니다.

단, OCR 후 바로 `회사`, `공장`, `부서`, `역할`을 분리해 저장해야 합니다. 이름/전화번호만 저장하면 가치가 낮습니다.

### 2.5 Off-the-record 메모

**실용성**: 매우 높음

산업재 영업의 핵심입니다.

예시:

- 설비팀 김과장이 실제 영향력 있음
- 구매팀은 단가만 보지만 생산팀은 납기를 더 중요하게 봄
- 경쟁사 제품 불량 경험 있음
- 결제는 늦지만 물량은 큼
- 안전 이슈 때문에 특정 소재 싫어함
- 공장장 교체 후 방향 바뀜

메모는 거래처, 담당자, 품목, 설비에 연결되어야 합니다.

## 3. 기획서가 놓친 산업재 영업 페인포인트

### 3.1 품목별 교체 주기

소모품, 공구, 부품은 재구매 주기가 있습니다. 이 주기를 잡으면 매출이 됩니다.

필요 기능:

- 품목별 마지막 납품일
- 예상 소진/교체일
- 재구매 알림
- 대체품 제안 메모

### 3.2 현장 사진과 설비 메모

공장 현장은 말보다 사진이 빠릅니다. 설비 명판, 고장 부위, 설치 상태, 경쟁사 제품 사진을 저장해야 합니다.

### 3.3 다중 담당자 관리

구매팀, 생산팀, 설비팀, 품질팀이 각각 다른 기준으로 의사결정합니다. 현재 고객 모델에 역할과 영향력을 추가해야 합니다.

### 3.4 다음 액션 관리

산업재 영업은 "견적 보내기", "샘플 전달", "테스트 결과 확인", "납기 확인" 같은 액션을 놓치면 바로 신뢰가 떨어집니다.

## 4. 우선순위 제안

### Phase 1

- 거래처/공장 카드
- 담당자 역할 관리
- 품목/납품 이력
- 현장 메모
- 빠른 검색
- 모바일 오프라인 조회

### Phase 2

- 다음 액션과 due date
- 교체 주기 알림
- 명함 OCR
- Excel Import

### Phase 3

- 현장 사진 첨부
- 샘플 테스트 기록
- 견적 후속 관리
- 저장 필터

### Phase 4 이후 또는 제외

- ERP 대체
- 자동 견적 생성
- 결제/세금계산서 처리
- 재고 시스템

## 6. 핵심 요약(TL;DR)

산업재 영업은 이 서비스의 핵심 타깃으로 적합합니다. 단, `고객/딜`만으로는 부족하고 **공장/담당자/품목/교체주기/다음액션**까지 내려가야 합니다.

반드시 보강할 기능:

1. 공장/부서/담당자 계층 구조
2. 품목별 납품 이력과 교체 주기
3. 다음 액션과 due date
4. 현장 사진/설비 메모
5. 담당자 역할과 영향력

이 persona 기준으로는 Phase 2에 `next_action`과 `교체 주기 알림`을 반드시 올려야 합니다.
