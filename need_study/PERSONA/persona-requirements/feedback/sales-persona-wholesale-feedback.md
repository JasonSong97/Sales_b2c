# 도매 영업 현장 관점 피드백 — 영업 관리 프로그램 서비스 기획서

> **피드백 작성자**: 최정훈 (6년차 식자재·주류·음료 도매 영업, 거래처 120곳 담당)
> **작성일**: 2026-05-22
> **대상 문서**: `01_서비스_기획서.md`

---

## 1. 전체 평가

**이 서비스는 "책상 있는 영업사원"용입니다.**
트럭+배송 겸업 도매 영업사원한테는 반은 맞고 반은 안 맞아요.

### 맞는 부분
- 거래처 120곳 정보, 사장님 메모, 외상 기록 같은 거 흩어져 있는 건 진짜 페인포인트
- 오프라인 조회, 명함 스캔, Off-the-record 메모는 정말 필요한 기능

### 안 맞는 부분
- **노트북에서 입력하고 핸드폰으로 조회한다는 전제가 안 맞음**
- 하루 종일 트럭 안이라 노트북 펼 일이 거의 없음
- 전부 핸드폰으로 입력해야 하는데, 양손 써야 하는 폼 입력은 현장에서 불가

## 2. 기능별 평가 ("트럭 운전하면서 한 손으로 쓸까?")

### 2.1 영업 데이터 관리 (CRUD)

**한 손 사용성**: △ (조회만 가능)

**조회 — OK**:
- 운전 중 "A 식당 사장님 외상 얼마더라?" 확인 가능
- 오전 8시 트럭 안에서 다음 거래처 확인 가능
- 오후 2시 신규 거래처 찾아갈 때 정보 조회 가능

**입력 — 불가**:
- 양손 써야 하는 폼 입력은 매장 들어가면서 못 함
- 한 손에 발주 전표, 한 손에 상품 박스 들고 들어감
- 저녁에 차에서 메모하려 해도 양손 필요하면 포기

**개선 제안 — 음성 메모 입력 필수**:
- 블루투스 이어폰 끼고 한마디로 끝나야 사용
- 예시 명령어:
  - "A 식당 외상 5만 원 추가"
  - "B 카페 오늘 주문 20만 원"
  - "C 편의점 다음 주 화요일 방문"
- 자동으로 해당 거래처 카드에 기록되어야 함

### 2.2 엑셀 Import/Export

**한 손 사용성**: X (트럭에서 안 씀)

- 엑셀 업로드는 PC 작업인데, 집에서 업무 안 함
- 회사 주문 시스템이 따로 있어서 엑셀 정리할 일도 없음
- 이직 시 거래처 목록 백업용 외에는 사용하지 않을 것

### 2.3 필터링 시스템

**한 손 사용성**: ○ (터치 3번 이내일 때)

**필요성**:
- "수원 지역만 보기", "외상 있는 곳만 보기" 진짜 필요
- 단, **한 손 터치 3번 안에** 필터 걸 수 있어야 함
- 메뉴 들어가서 체크박스 누르는 방식이면 안 씀

**개선 제안**:
- 홈 화면에 **"즐겨찾기 필터" 버튼 3개** 고정
  - 예: [오늘 루트] [외상 있는 곳] [신규 개척 후보]
- 필터 결과를 **지도 위에 핀으로** 표시 (다음 동선 짤 때 유용)

### 2.4 거래처 정보 관리 (명함 OCR)

**한 손 사용성**: △

**명함 스캔 OCR — 좋음**:
- 신규 개척할 때 사장님 명함 받으면 바로 찍어서 저장
- 단, 명함보다 **매장 사진**을 더 많이 찍음
  - 예: "이 상가 2층 새 카페 오픈 예정"
  - **명함이랑 매장 사진 같이 저장**되면 좋겠음

**수정 화면 우려**:
- 수정 화면이 복잡하면 안 씀
- OCR 잘못 읽은 거 고치려고 폼 여러 개 터치하면 종이 수첩이 나음
- 수정은 한두 번 터치로 끝나야 함

**Off-the-record 메모 — 진짜 필요**:
- "B 사장님 아들이 고3이라 3월부터 기분 안 좋음"
- "C 사장님 금요일마다 골프, 목요일 밤에 주문 전화하면 잘 받아줌"
- 회사 시스템에 못 적는 내용
- 단, **음성 메모 녹음 + 텍스트 변환** 필수 (양손 타이핑 불가)

### 2.5 인증

**필수. 단, 자동 로그인 되어야 함.**
- 트럭에서 앱 켤 때마다 비밀번호 치면 안 씀
- 지문인식 OK

## 3. 도매 영업 현장 페인포인트 (기획서가 놓친 것)

### 3.1 음성 입력이 없다 (CRITICAL)

**제일 큰 문제. 하루 종일 한 손이 비지 않음.**

현장 상황:
- 운전 중: 핸들 잡고 있음
- 트럭 내릴 때: 짐 들고 내림
- 매장 들어갈 때: 발주서+상자 들고 들어감

**필수**: 블루투스 이어폰 + 음성 명령
- "A 식당 외상 5만 원 추가"
- "B 카페 오늘 주문 20만 원"
- "C 편의점 다음 주 화요일 방문"
- "D 사장님 아들 고3, 예민함"

### 3.2 오늘 동선/루트 관리 기능이 없다 (CRITICAL)

**하루에 120곳 중 15~20곳 방문. 그날그날 동선이 다름.**

필요한 기능:
- "오늘 가야 할 곳" 리스트
- 지도에서 거래처 위치 + 순서
- 방문 완료 체크

**개선 제안**: 지도 연동 + 오늘 방문 체크리스트
- 홈 화면에 "오늘 방문" 탭
- 지도에 핀으로 표시 + 최적 동선 제안
- 방문 완료하면 체크 → 다음 거래처로

### 3.3 외상/미수금 관리가 약하다 (IMPORTANT)

**거래처별로 외상 언제 받을지, 얼마나 밀렸는지 관리해야 함.**

현재 기획서 문제점:
- 딜(deal) 테이블만 있고 **외상 관리는 없음**
- 영업 건(deal)과 외상(미수금)은 다른 개념

필요한 기능:
- 거래처별 외상 현황 (총액)
- 회수 예정일
- 외상 추가/회수 이력
- 알림: "오늘 A 식당 외상 회수 예정일"

### 3.4 주문 패턴 기록이 없다 (IMPORTANT)

**사장님별 주문 패턴을 알아야 효율적으로 관리 가능.**

필요한 기능:
- 거래처별 주문 주기 (예: 매주 화·금)
- 평균 주문 금액
- 최근 주문 이력
- "이번 주 아직 주문 안 들어온 곳" 알림

예시:
- "A 식당은 매주 화·금 주문, 평균 15만 원"
- "B 카페는 월초에 몰아서 주문, 평균 80만 원"

### 3.5 신규 개척 관리가 약하다 (NICE TO HAVE)

**오후에 상가 돌면서 "여기 새 카페 생겼네" 하고 사진 찍어두는데, 정리할 공간이 없음.**

필요한 기능:
- "신규 후보" 상태 추가
- 위치 저장 (지도 핀)
- 매장 사진 여러 장 저장
- 메모: "2층 공사 중, 5월 오픈 예정"

## 4. 현재 사용 중인 도구 vs 이 서비스

| 현재 도구 | 용도 | 불만 | 이 서비스로 대체 가능? |
|----------|------|------|---------------------|
| 회사 주문 시스템 (PDA/앱) | 주문 입력, 정산 | 주문만 됨, 영업 메모 안 됨 | ❌ 대체 불가 (본사 강제) |
| 카톡 (150개 채팅방) | 사장님 소통, 주문 | 주문·메모·안부 다 섞임 | ❌ 대체 불가 (사장님들이 이미 사용) |
| 종이 수첩 (대시보드) | 거래처 패턴, 외상, 성향 | 차에서 불편, 비에 젖음 | ✅ **음성 메모 + 거래처 카드** |
| 갤러리 | 매장 사진, 영수증, 명함 | 정리 안 됨 | ✅ **명함 스캔 + 매장 사진** |
| 카카오맵 | 길찾기 | 거래처 정보와 분리 | ✅ **지도 연동 + 오늘 루트** |
| 종이 거래명세서 | 일일 정산 | 분실, 글씨 흘려 씀 | △ 부분 대체 |

## 5. 우선순위 제안

### Phase 1 수정 제안

**추가**:
- 음성 메모 입력 (STT)
- 한 손 조작 최적화 (버튼 크기, 터치 간소화)
- 자동 로그인 (지문인식)

**이동**:
- 엑셀 Export → Phase 2로 (현장에서 안 씀)

### Phase 2 수정 제안

**추가**:
- 지도 연동 (거래처 위치 핀)
- 오늘 방문 루트 체크리스트
- 외상/미수금 관리 기능

**이동**:
- PDF 요약 출력 → Phase 4로 (현장에서 안 씀)

### Phase 3 유지

- 명함 스캔 OCR — Good
- Off-the-record 메모 — Good

### Phase 4 추가

- 주문 패턴 자동 집계
- 외상 회수 알림
- 신규 후보 관리

## 7. 핵심 요약 (TL;DR)

### 타겟 적합도

| 타겟 | 적합도 | 이유 |
|------|--------|------|
| 책상 있는 B2B 영업 (IT·금융·컨설팅·제약) | ✅ 적합 | 노트북→모바일 동기화 전제에 맞음 |
| 트럭+배송 도매 영업 (식자재·주류·음료) | ⚠️ 조건부 | 음성입력+한손조작+지도연동 추가 시 적합 |
| 프리랜서·1인 영업 | ✅ 적합 | 개인용 관리 도구 필요 |

### 이 앱을 쓸 수 있는 3가지 조건

1. **음성 메모 입력** — 블루투스 이어폰 끼고 한마디로 끝나야 함
2. **한 손 조작** — 터치 3번 이내로 원하는 정보까지 도달
3. **오프라인 동작** — 지하 주차장·터널에서도 조회 가능 (기획서에 있음, Good)

### 추가하면 결제까지 하는 기능

4. **지도 연동** — 오늘 방문 루트 + 거래처 핀
5. **외상 관리** — 거래처별 미수금 현황 + 회수 알림

### 실용성 테스트 제안 (베타 시)

1. 운전 중 블루투스 이어폰으로 메모 입력 가능한가?
2. 한 손으로 거래처 조회 → 메모 추가까지 3초 안에 가능한가?
3. 지하 주차장에서 오프라인 조회 되는가?
4. 트럭 내려서 짐 들고 앱 켜기까지 몇 초?
