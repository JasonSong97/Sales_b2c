### 영업 미팅 회의록 AI 요약

: 영업 미팅을,  [ 1) 녹음 후 텍스트 추출 /  2) 텍스트 입력 ]⇒  AI 가 자동으로 회의록을 만든다.

**화면구성**  (위치: AI 회의록 탭 )
1.  생성된 회의록 목록 → 회의록 상세페이지   
2.  회의 음성노트 버튼 → 회의 음성 녹음 + AI 요약 회의록 생성     
3.  회의 노트 버튼 → 회의노트 (사용자가 직접 입력) + AI 요약 회의록 생성

**기능**
1. 회의 노트 
 - 회의 음성노트 : 녹음 >  녹음 에서 텍스트 추출 > AI 회의록 자동 생성 
 - 회의 노트 : 회의 내용 입력 >  노트내용으로 AI 회의록 자동 생성

1. AI 회의록 생성  : 생성한 AI 회의록 확인 후 (수정도 가능) 저장 
( 대략적 회의록 구성 : 날짜, 회사, 담당자 , 부서, 품목, 진행단계, 상세내용, 향후계획, 필요액션)
2. 영업 딜 과 연동
 회의록 저장 후, 영업딜과 연동 버튼 
  > 영업 딜과 연동 (영업딜 - 활동기록 탭 > 해당 날짜로 활동기록 로그 생성, 내용 - 회의록 링크)

---

주 목표 : `회의 내용의 맥락을 저장하는 것이 목표`, 화자 분리가 목표가 아님
