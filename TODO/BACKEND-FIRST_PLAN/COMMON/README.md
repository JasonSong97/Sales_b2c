# Backend First COMMON

## 1. 목적

`COMMON`은 Backend 우선 구현 중에도 FE와 BE가 함께 봐야 하는 계약을 모아 둔다.

이 계획에서는 새로운 API 계약을 중복 작성하지 않고, 기존 `MVP-STARTER_PLAN`의 API/DB/goal 명세를 정본으로 참조한다. 이 폴더는 Backend 우선 실행 순서, 완료 기준, 검토 결과를 관리한다.

## 2. 문서

| 문서 | 역할 |
|---|---|
| `USER-FLOW.md` | Backend 구현이 지켜야 하는 사용자 흐름 요약 |
| `GOAL-WORK-ORDER.md` | Backend 우선 `/goal` 작업 순서 |
| `PLANNING-REVIEW.md` | 구현 전 검토 결과 |
| `API-SPEC/README.md` | 기존 API 계약 참조 방식 |
| `GOAL-SPECS/README.md` | 기존 goal 상세 명세 참조 방식 |
