# AGENT

This folder contains planning, architecture, convention, and decision documents for the project.

## Folders

- `PLANNING`: product plans, PRD, MVP scope, user flows
- `CONVENTION`: coding, naming, comment, logging rules
- `ARCHITECTURE`: frontend, backend, admin, testing, future mobile architecture rules
- `DECISIONS`: confirmed decisions and unresolved decision logs
