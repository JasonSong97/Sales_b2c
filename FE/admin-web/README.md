# admin-web

Admin web app.

## Initial Scope

- User list and detail
- Global deal list
- Global company list
- Global contact list
- Global product list
- Per-user deal/company/contact/product views
- Later: manual bank-transfer payment status management

Admin APIs use `/admin/api/*`.

## Local Setup

Prerequisite: Node.js 24 LTS must be active.

```bash
pnpm install
pnpm run dev
```

Local URL: `http://localhost:5174`

Vercel project root: `FE/admin-web`
